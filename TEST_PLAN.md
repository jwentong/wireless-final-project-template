# 测试计划 (TEST_PLAN.md)

## 1. 单元测试计划
* **源编码模块**：输入中英文混合文本，编码再解码，断言恢复文本与输入一致。
* **调制解调模块**：生成随机奇数个比特，测试 padding 补0逻辑，断言无噪声下解调完全一致。
* **同步模块**：人为在 QPSK 符号前插入 50 个 0 符号，测试是否能准确返回 index=50。

## 2. 端到端集成测试计划
* **高信噪比测试** (SNR = 20dB)：检查 `received.txt` 与 `Test.txt` 的 MD5 校验和是否完全一致。
* **低信噪比测试** (SNR = 0dB)：系统不崩溃，BER 大于 0，`checksum_pass` 应为 false。
